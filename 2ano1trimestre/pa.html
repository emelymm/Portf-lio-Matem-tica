Progressão Aritmética

   A progressão aritmética (PA) é uma sequência numérica em que cada termo é igual à soma do termo anterior com um número fixo, chamado de razão. Por exemplo, a sequência (1, 4, 7, 10, 13, ...) é uma progressão aritmética com razão 3, já que para obter cada termo a partir do anterior, é necessário adicionar 3. Da mesma forma, a sequência (10, 7, 4, 1, -2, ...) também é uma progressão aritmética com razão -3, já que para obter cada termo a partir do anterior, é necessário subtrair 3.

   Assim, podemos descobrir a razão de uma PA subtraindo um número com o seu anterior, veja abaixo:


( a1, a2, a3, a4, ..., an-1, an )


   Nessa PA, an é o último termo e an-1 é o penúltimo, isto porque, como podemos ver ao lado, o penúltimo número é o último menos a razão.

a2 = a1 + r (razão)

...

an = an - 1 + r


Razão: r = an - an - 1

   Também podemos classificar a progressão, podendo ser crescente caso r > 0, decrescente caso r < 0, ou até mesmo costante, quando r = 0. Por exemplo, uma PA ( 4, 4, 4, 4 ) é costante, já que o 4 será sempre somado com a razão 0.





Fórmula do Termo Geral

   Para descobrirmos um certo termo de uma progressão aritmética utilizamos a fórmula do termo geral, que nos permite descobrir esse termo através do primeiro termo e da razão dessa progressão. Veja a fórmula abaixo:

an = a1 + ( n - 1 ) r


OBS.: Ao passar de a1 para an avançamos ( n - 1 ) termos.

  • a1 -> 1° termo da progressão;
  • an -> termo geral da progressão;
  • n -> número de termos da progressão ou posição do termo que deseja saber;
  • r -> razão.


EX.: Temos uma PA ( 3, 5, 7, ... ) e gostaríamos de descobrir o 50° termo dessa progressão.

RESPOSTA: Primeiro precisamos observar os valores que já temos. Sabemos que a1 = 3 e que a razão é r = a2 - a1 = 5 - 3 = 2. O enunciado nos informa que precisamos saber o 50° termo, logo n = 50. Então, tendo essas informações basta utilizarmos a fórmula.

a50 = 3 + ( 50 - 1 ) 2

a50 = 3 + 98

a50 = 101



   Assim como outras funções, também podemos descobrir outros termos além do termo geral através dessa fórmula.

EX.: Tenho uma progressão em que a razão é -3 e sei que a1 = 5 e que o último termo é -22, preciso descobrir quantos termos essa sequência tem.

RESPOSTA: Sabemos que n pode ser tanto o número do termo que desejamos saber, quanto o número de termo de uma PA, isso porque, se soubermos o último termo, podemos utilizar ele como termo geral e quando formos descobrir o n saberemos sua posição e consequentemente o número de termos dessa PA. Então, sabemos que a1 = 5, r = -3, an = -22, e precisamos descobrir n.

-22 = 5 + ( n - 1 ) ( -3 )

-22 = 5 - 3n + 3

3n = 8 + 22

n = 10



   Também podemos descobrir os termos de uma progressão aritmética levando em consideração apenas a razão e aquilo de que a razão vai ser um número menos o seu anterior.


EX.: Temos uma PA ( x2, (x + 2)2, (x + 3)2 ), sabendo isto, qual é p valor de x?

RESPOSTA: Sabemos que a1 = x2, a2 = (x + 2)2 e a3 = (x + 3)2. Também sabemos que razão = a2 - a1 = a3 - a2. Então, basta substituirmos os termos dessa ideia da razão e encontrarmos o x.

( x + 2 )2 - x2 = ( x + 3 )2 - ( x + 2 )2

x2 + 4x + 4 - x2 = x2 + 6x + 9 - ( x2 + 4x + 4 )

4x + 4 = 2x + 5

2x = 1

x = 1/2





Interpolação Aritmética

   Interpolação é inserção, logo na interpolação aritmética iremos inserir termos em uma progressão aritmética. Na interpolação chamamos a1 e an de extremos e os termos que iremos inserir entre eles serão os meios, os quais identificaremos como k.

( a1, k1, k2, k3, k4, k5, an )


EX.: Precisamos interpolar 5 meios aritméticos entre 1 e 2.

RESPOSTA: Sabemos que a1 = 1 e que an = 2. Além disso, se precisamos interpolar 5 meios e temos 2 números, o tamanho dessa progressão é n = 7, então a7 = 2. Para descobrirmos os outros números precisamos saber a razão dessa progressão, então através da função do termo geral iremos descobrir a razão.


an = a1 + ( n - 1 ) r

2 = 1 + 6r

-6r = -1

r = 1/6

k1 = 1 + 1/6 = 7/6

k2 = 1 + 7/6 = 8/6 : 2 = 4/3

k3 = 1 + 8/6 = 9/6 : 3 = 3/2

k4 = 1 + 9/6 = 10/6 : 2 = 5/3

k5 = 1 + 10/6 = 11/6


Resultado Final : PA ( 1, 7/6, 4/3, 3/2, 5/3, 11/6, 2 )




Soma dos Termos de uma PA finita

   Como o próprio título diz, a soma dos termos de uma PA finita será a soma dos termos de uma progressão aritmética até um certo termo n. Para isso utilizamos a seguinte fórmula:

Sn = ( a1 + an ) n / 2


EX.: Qual é a soma dos 25 primeiros termos de uma PA ( 1, 7, 13, ... )

RESPOSTA: Sabemos que a1 = 1, n = 25 e r = 6, portanto, precisamos descobrir o valor de a25 para poder calcular a soma dos 25 termos.


an = a1 + ( n - 1 ) r

an = 1 + ( 25 - 1 ) 6

an = 145

Sn = ( a1 + an ) n / 2

S25 = ( 1 + a145 ) 25 / 2

S25 = 3650/2 = 1825.
